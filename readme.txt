== Description ==

Content Sync is a powerful WordPress plugin that allows you to synchronize content across multiple websites, saving you significant time in content management workflows.

With Content Sync, you can:

* Synchronize posts, pages, and custom post types across multiple WordPress installations
* Manage content from a central location
* Keep multiple sites in sync automatically
* Review and approve content before syncing
* Track synchronization history and status

Perfect for agencies managing multiple client sites, multi-site networks, or anyone who needs to keep content consistent across multiple WordPress installations.

== Installation ==

1. Upload the `contentsync` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin settings to connect your sites
4. Start syncing your content!

== Frequently Asked Questions ==

= How do I connect multiple sites? =

Configure the plugin settings to add remote sites and establish connections between them.

= What types of content can be synced? =

The plugin supports syncing posts, pages, and custom post types. Check the plugin settings for specific options.

= Is my data secure? =

Yes, all synchronization uses secure REST API connections. Make sure to configure proper authentication for your sites.

= Can I review content before syncing? =

Yes, the plugin includes a review system that allows you to approve or reject content before it's synced to target sites.
